# Look At Butterfly

一個互動式網頁專案，包含一隻會跟隨滑鼠但又會無聊飛走的蝴蝶，以及一對會注視著蝴蝶的眼睛。

## 功能特色

- 🦋 **互動蝴蝶**:
  - 預設在畫面中隨機飛行 (Wandering)。
  - 當滑鼠在畫面移動時，會被滑鼠吸引 (Following)。
  - 如果滑鼠停止移動超過設定時間，蝴蝶會感到無聊並飛走 (Fleeing)。
  - 具備平滑轉向與翅膀拍動動畫。

- 👀 **注視眼睛**:
  - 畫面中央有一對眼睛，眼珠會隨時盯著蝴蝶的位置移動。

- 🎛️ **控制面板**:
  - 可調整蝴蝶飛行速度 (Speed)。
  - 可調整蝴蝶感到無聊前的等待時間 (Boredom Delay)。

## 安裝與執行

由於本專案依賴 NPM 套件，請依照以下步驟執行：

1. **安裝依賴**:
   ```bash
   npm install
   ```

2. **啟動開發伺服器**:
   ```bash
   npm run dev
   ```

3. **開啟瀏覽器**:
   前往終端機顯示的網址 (通常是 `http://localhost:5173`)。

## 技術棧

- Vue 3 + TypeScript
- Vite
- Tailwind CSS
- VueUse (用於滑鼠追蹤與視窗尺寸)
- Radix Vue (用於控制面板滑桿)
